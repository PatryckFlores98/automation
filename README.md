

**References**
- [Cypress](https://www.cypress.io/)

## Setup - Windows

**Requerimentos**
- [Node.js](https://nodejs.org/en/)
- Install the Cypress on the folder
```
$ npm install
```
- Download and install [Visual Studio Code](https://code.visualstudio.com/).


## Executing Tests

**Requirements**
- Via terminal:
```
$ npx cypress run
```
- Via interface:
```
$ npx cypress open
```
